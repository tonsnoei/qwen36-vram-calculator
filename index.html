For classical models: KV memory = 2 × L_attn × Nkv × Dkv × S × B × Cb. Hybrid models (Qwen3.6) only count the classical attention layers (16/64 for 27B, 10/40 for 35B-A3B); the linear-attention layers carry a small batch-proportional state instead. MoE models hold all expert weights resident in VRAM, but activation memory scales with active params (3B for 35B-A3B). Q4 uses 0.55 b/p (~10% metadata overhead). Specs from official Qwen3.6 model cards and Gemma-2 paper.
